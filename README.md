
## 🎯 Research Questions
- **RQ1**: Can black-box membership inference attacks successfully infer training membership in federated learning?
- **RQ2**: How does the number of federated clients affect attack success?
- **RQ3**: How does model complexity influence privacy leakage?

## 🛠️ Methodology
- **Dataset**: CIFAR-10 partitioned across multiple clients
- **Federated Learning**: Simulated using FedAvg algorithm
- **Attack Model**: Binary classifier (logistic regression) to distinguish member vs non-member samples

## 📊 Key References
- Shokri et al. (2017) - Membership inference attacks against machine learning models
- McMahan et al. (2017) - Communication-efficient learning from decentralized data
- Nasr et al. (2019) - Privacy analysis of deep learning
